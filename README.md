# TableReserve Bot

Telegram bot for single-restaurant table reservations: book available slots, get confirmation codes and reminders, and reschedule or cancel via inline buttons; owner dashboard included.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
